サンプルの実行方法

1. sample.HsqlDbStartクラスのmainメソッドを実行してhsqlデータベースを起動します。
　※　他のサンプルの実行時に既にhsqlデータベースが起動されている場合は、
　　　一度停止してから起動し直してください(読み込んでいるデータが異なるため)

2. sample.MyBatis3Mainクラスのmainメソッドを実行すると、MyBatis3の機能のサンプルが実行されます。

3. sample.SpringMainクラスのmainメソッドを実行すると、MyBatis3とSpringの連携(Mapperインタフェースを使用しない)のサンプルが実行されます。

4. sample.SpringMapperMainクラスのmainメソッドを実行すると、MyBatis3とSpringの連携(Mapperインタフェースを使用する)のサンプルが実行されます。
